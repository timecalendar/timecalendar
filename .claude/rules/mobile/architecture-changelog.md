# Rule changelog — TimeCalendar mobile

One of the [five living artifacts](./architecture.md#the-five-living-artifacts): a dated,
**append-only** (newest last) log of every change to the rules — the [Architecture Book](./architecture.md)
hub + topical files or the [Definition of Done](./definition-of-done.md). Per migration-approach §7,
the act of changing a rule is itself recorded here. Each entry: date · change-slug · what moved · why ·
pointer link. **Backfilled** below the marker (seven pre-changelog eras, authored retroactively from the
landed changes); everything below the live marker is appended live.
